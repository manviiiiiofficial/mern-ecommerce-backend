# MERN Ecommerce Backend

This is a backend project for an ecommerce application built using Node.js, Express, and MongoDB.

## Features
- User registration
- Product management
- Cart system
- API routes for ecommerce operations

## Technologies Used
- Node.js
- Express.js
- MongoDB

## How to Run

1. Clone the repository
2. Run npm install
3. Start the server with:
   npm start
